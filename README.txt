# _Immortal Media Books_

#### _Here is a basic bookstore site that includes different user permissions and special book review features_

#### By _**Tim Bourgault**_

## Description

* _Bookstore site that includes an About Page, a Location page (complete with Google Map), a Contact page with site feedback contact form, a special coupon for Authenticated users, a Book Reviews page that can be reached through the navigation menu which sorts the book reviews by the number which they are rated.  The site makes use of the features, strongarm, and views modules in Drupal to track and organize components to easily move and save.


## Setup/Installation Requirements

* _Clone the program from it's repository at GitHub.com by searching for the user name Timothy-Bourgault and then looking for bookstore repository_
* _Open MAMP and go into the preferences menu.. Under the Weg Server tab, change the document root folder to the upper level folder of the cloned repository_
* _Start Apache and MySQL servers_
* _Click Open WegStart page to access phpMyAdmin under the tools dropdown_
* _Once inside phpMyAdmin, select the import tab and then click the Choose File to browse your computer for the database backup inside the cloned repository_
* _Once the database backup has been imported you can then access the site by directing your browser to localhost:8888_
